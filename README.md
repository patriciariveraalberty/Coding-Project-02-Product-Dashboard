# Coding Project 02 - Product Dashboard

## Description

This project is an interactive product dashboard built with HTML, CSS, and JavaScript. It retrieves product information from the Store Products API and dynamically displays products in a responsive dashboard.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Store Products API

## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in your web browser.

## Features

- Fetches product data from an external API
- Dynamically displays products
- Responsive product cards
- Modern user interface
- Error handling for API requests

## What I Learned

This project helped me strengthen my understanding of working with APIs, asynchronous JavaScript using `fetch()`, DOM manipulation, and dynamically displaying data in a responsive web application.
